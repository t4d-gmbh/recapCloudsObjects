### Hands-on: Object Storage in the GUI

:::{admonition} Terminology
:class: note margin
In Swift, buckets <i class="fa-solid fa-bucket"></i> are called **containers**. Same concept, different name.
:::

1. Open the **Web GUI** → navigate to **Object Store → Containers**
2. Create a new **container** (= bucket <i class="fa-solid fa-bucket"></i>)
3. Upload a file. E.g. the `.sif` container from the `exoHelloWorld` project
